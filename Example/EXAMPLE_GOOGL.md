## 二、业务详情

**本章要点**：Alphabet 是"成熟广告现金牛（~72% 收入）+ 超高速云业务（~21%，Cloud）+ 期权组合（Other Bets/投资）"的三层结构；Q2'26 的关键变化是 Cloud 从"追赶者"变为"AI 基建赢家"，收入结构在实时重构。

商业模式：广告（搜索意图流量变现 + YouTube 注意力变现 + 联盟网络）、云计算（GCP 基础设施/AI 平台 + Workspace 订阅）、订阅与设备（YouTube Premium/TV、Google One、Pixel）、以及 Waymo 等前沿业务。收入确认以广告投放（即时）与云消耗/订阅（按用量/期间）为主。

## 三、业务与竞争分析

**本章要点**：搜索护城河（意图数据 + 分发 + 广告主网络）迄今经受住生成式 AI 的正面冲击；Cloud 凭 TPU/全栈 AI 基础设施从行业第三变成增速第一；真正的长期威胁不是"搜索被替代"，而是"答案式交互压缩商业查询量 + 军备竞赛资本化竞争优势"。

行业与周期：全球数字广告约 $700B+ 量级、个位数至低双位数增长（成熟）；云基础设施 $400B+ 量级、AI 驱动再加速（成长中段）。Alphabet 横跨两者，当前处于 AI 基建投入周期的陡峭段（行业性，非公司独有——微软、Meta、亚马逊同步军备竞赛）。

**护城河评分表**：

| 护城河来源 | 是否具备 | 强度 | 依据 |
|---|---|---|---|
| 无形资产（数据/算法/品牌） | 是 | 强 | 20+ 年意图数据、DeepMind/Gemini 前沿模型自研、"Google=搜索"心智 |
| 转换成本 | 是 | 中→强 | 广告主工具链/归因体系；Cloud 工作负载迁移成本随 AI 合同长约化在上升 |
| 网络效应 | 是 | 强 | 广告主↔用户双边市场；YouTube 创作者生态；Android/Chrome 分发 |
| 成本优势 | 是 | 强 | 自研 TPU + 全球数据中心规模：单位算力成本低于依赖外购 GPU 的对手（Cloud 利润率 35.5% 是证据） |
| 规模效应 | 是 | 强 | 研发 $69B/年（TTM）可摊薄在 $446B 收入上 |
| **综合判断** | | **宽** | **但注意定量趋势反向**：EPV/净资产 3.2x→2.1x（第六章），护城河"宽度"未变、"每单位资本的护城河产出"在稀释 |

增长引擎（3–5 年）：Cloud/AI 基建（最大单一引擎）、搜索 AI 化变现（成本每查询上升 vs 变现增强的赛跑）、YouTube（TV 大屏 + Shorts + 订阅）、Waymo（期权，尚不进模型）。

主要风险与证伪点：(1) AI-native 入口（ChatGPT 类）分流商业查询——当前数据未见，但属"慢变量"；(2) 反垄断救济措施的结构性影响（美国搜索分发案与广告技术案的救济阶段进展，**未获取到**截至本报告日的最新裁定细节，标注为待核验）；(3) capex 军备竞赛长期化侵蚀 FCF；(4) TPU/供应链地缘风险。

## 四、管理层、治理与资本配置计分卡

治理：双层/三层股权（B 类 10 票，创始人 Page/Brin 合计控制多数投票权；C 类 GOOG 无投票权）。小股东对重大资本决策（如 capex 规模）无实质约束力——在 AI 支出争议下，这一结构性风险的"价格"正在上升（我的判断）。管理层：Sundar Pichai（CEO，2015 至今）执行记录稳定；DeepMind/AI 整合成效显著。

## 六、估值（多方法交叉验证）

**本章要点**：五种方法全部由 `scripts/dcf.py` 执行（假设 JSON：`googl_valuation.json`，禁止心算）；保守方法（EPV $120、EVA $108）与乐观锚（Morningstar $433）相差 4 倍，本身就是本案信息量——价值几乎全部押在"AI capex 回报"这一个变量上。综合区间 **[230, 320]**，现价 317.69 落在**上限附近**。

### 6.1 反向 DCF + PVGO：现价假设了什么（开篇框架）

以现价 $317.69 反解（WACC 9%、永续 g 3%、5 年建设期 FCF 路径 [60, 80, 105, 135, 165]）：

- 现价隐含**稳态首年 FCF ≈ $310B/年**（TTM 实际 $53B，FY25 高点 $73B——即要求 FCF 在稳态达到当前的 4–6 倍）；
- 换算：@FCF 率 20% → 需营收 $1,548B（5 年 CAGR 27%）；@24% → $1,290B（22%）；@28% → $1,106B（19%）。
- **Base rate 对照**：$450B 收入基数上维持 5 年 19–27% 增速，处于历史分布第 90+ 分位——历史上没有任何公司在此规模做到过 20%+（base-rates.md §1 规模惩罚）。需要说明：本反向口径为"5 年后即进入稳态"，若允许 6–10 年更长的高增长跑道（如三情景的衰减期结构），隐含要求会缓和至 ~15% 一档——市场实际定价介于两者之间。
- **PVGO 分解**：零增长价值 = 常态化 EPS $9.9 ÷ 9% = **$110/股**；现价的 **65%（$208/股）在为未来增长付费**。
- 结论句：**市场不是在给"全球最大广告公司"定价，而是在给"AI 时代的算力+入口双寡头"定价。**

### 6.3 三要素 / EPV（Graham–Greenwald）

- 常态化 NOPAT $121B ÷ WACC 9% → EV $1,344B + 净现金 $130B = 权益 $1,474B → **EPV = $120/股**。
- **护城河财务验证（本案关键定量发现）**：EPV/调整后净资产 = 2.30x（宽护城河区间），但多年趋势 **FY23 2.84x → FY24 3.21x → FY25 2.87x → TTM 2.10x**——特许经营价值/单位资本连续两年压缩。这与第三章"宽护城河"定性结论**方向一致但强度衰减**：护城河没有变窄，而是公司在把每一美元利润配到回报更低的资产上。
- Franchise 成长价值（g 5%、ROIIC 21.5%）：**$200/股**。买点阶梯：底价 $52 ｜ EPV $120 ｜ 成长调整 $200 ｜ **现价 $318 > 成长调整**——在 Greenwald 框架下，现价已为"成长之外的东西"（AI 期权）付费 $118/股。

### 6.4 EVA / 剩余收益

投入资本 $422B、NOPAT $121B → 存量 ROIC **28.7%**、超额利差 +19.7pp；15 年利差线性衰减假设下，剩余收益价值 = $422B + PV(EVA) $766B + 净现金 = **$108/股**。
**自洽检验**：内生增长上限 = 再投资率 55% × ROIIC 21.5% = **11.8%**——我的 Base 情景前两年 15% 增速已超出内生上限，与"TTM 净发债 $70B"的现实互相印证（增长正在动用外部资本），假设自洽。EVA 法的 15 年利差归零假设对 Alphabet 这类公司偏严苛，本报告将其作为**下沿锚**而非中枢。

### 6.5 相对估值（合理倍数纪律）

常态化 EPS ≈ $10.4（NOPAT $121B + 经常性净利息 ≈$6B，÷12.245B 股）。应得 P/E =(1−g/ROIIC)/(r−g)：g 5%/ROIIC 21.5% → **19.2x**；乐观档 g 6%/ROIIC 25% → **25.3x**。对应每股 **$199–263**。当前常态化 P/E 30.6x，高于合理倍数上沿 21%——市场支付的溢价即 AI 期权定价。自身历史前瞻 P/E 中枢（18–25x）亦低于当前。
