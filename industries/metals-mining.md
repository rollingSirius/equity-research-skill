# 金属/矿业行业深度研究附录

适用于基本金属、贵金属、电池金属、铁矿、煤炭矿商及冶炼加工。特征：价格接受者 + 资源枯竭性资产，价值 = 商品价格假设 × 成本曲线位置 × 储量寿命。

## 1. 研究原则

- 矿商没有定价权，研究的一半是研究商品本身：供需平衡表、成本曲线（90 分位现金成本是长期价格锚）、库存周期。
- **NAV 是主框架**：逐矿山 DCF（矿山寿命有限，不许无限永续）加总 − 净债 − 关闭/复垦负债。
- 成本曲线位置决定生死：一/二分位成本矿商可穿越周期，四分位是价格期权。
- 贵金属矿商另看 AISC 与储量替代率；勘探型公司是期权组合不是现金流资产。

## 2. 价值驱动与 KPI

```text
NAV = Σ 矿山[产量 × (价格 − AISC) × 寿命折现] + 勘探期权 − 净债 − 复垦负债
```

| 指标 | 要点 | 陷阱 |
|---|---|---|
| 产量指引 | 分矿山、分品位 | 品位下滑被产量掩盖 |
| **AISC / C1 成本** | 对照行业成本曲线分位 | 副产品抵扣口径差异大 |
| 储量/资源量 | P&P vs M&I&I；储量寿命 | 价格假设抬高即"创造"储量 |
| 资本开支 | 维持性 vs 增长性拆开 | 维持性 capex 低报虚增 FCF |
| 套保头寸 | 比例与价格锁定 | 牛市被套保封顶 |

## 3. 估值

- **NAV（P/NAV）为主**：商品价格用远期曲线 + 长期激励价格（incentive price，支撑新增供给的价格）双锚，禁止用现货永续化。
- 中周期盈利 × EV/EBITDA 中枢做交叉验证；现货情景/长期价格情景分列。
- 价格敏感性必做：商品价格 ±20% 对 FCF 与 NAV 的弹性（高杠杆矿商弹性可达 3–5 倍）。
- 政治/税收风险按矿山所在国单独折价（资源国有化、暴利税历史）。

## 4. 数据源与财报模式

1. 技术报告（NI 43-101/JORC）是储量与成本的一手来源；年报储量表逐年对账。
2. 商品数据：LME/COMEX/SHFE 库存与价格、WoodMac/CRU 成本曲线（付费）、USGS、各国海关。
3. 财报重点：实现价格 vs 基准价差、AISC 环比、品位趋势、资本开支超支、股息政策（矿商喜欢周期顶分红顶点收购）。

## 5. 红旗与输出要求

- 红旗：储量价格假设逐年抬高、维持性 capex 连年低于折旧、周期顶并购、复垦负债折现率激进、社区/环保冲突升级。
- 输出：成本曲线定位图（文字版：本公司 AISC 处于全球第 X 分位）+ 分矿山 NAV 表 + 商品价格敏感性表必备。
