# 汽车/EV 行业深度研究附录

适用于整车（OEM）、EV 新势力、零部件、经销商、电池链整车侧。特征：重资产低毛利强周期 + 电动化/智能化重构价值链，传统框架与成长框架必须分开使用再对账。

## 1. 分类与研究原则

- **传统 OEM**：周期价值股框架——中周期盈利、产能利用率、净现金（含金融子公司要拆开：工业实体 vs 金融实体分别估值）。
- **EV 纯玩家**：成长框架——单车经济（ASP − BOM − 制造费用）、爬产斜率、现金 runway；盈利前公司用反向 DCF + 情景。
- **零部件**：看客户/平台绑定、单车价值量（CPV）提升逻辑（电动化受益 or 受损）、年降压力。
- 智能驾驶/软件收入（FSD 类）：期权性资产，单独定价，给概率不给确定性。

## 2. 价值驱动与 KPI

```text
整车价值 = 销量 × (ASP − 单车成本) × 周期调整 + 金融子公司 + 软件期权
```

| 指标 | 要点 | 陷阱 |
|---|---|---|
| 销量/交付 | 分地区分车型；批发 vs 零售口径 | 压库（渠道库存天数）冲量 |
| ASP 与单车毛利 | 剔积分/补贴后的真实毛利 | 监管积分收入美化毛利率 |
| 产能利用率 | 盈亏平衡点销量 | 新厂爬坡期亏损归为"一次性" |
| 订单/交付周期 | 需求领先指标 | 可退订金的订单注水 |
| 电池成本 | $/kWh 趋势与技术路线 | 原材料联动定价滞后 |
| 金融子公司 | 渗透率、残值风险、坏账 | 促销性金融补贴藏在金融子公司 |

## 3. 估值

- 传统 OEM：中周期 EPS × 历史中枢 P/E（通常 6–10x）+ 工业净现金调整；EV/Sales 对周期股无意义。
- EV 成长期：反向 DCF 为主框架（现价隐含销量 × 单车利润，对照全球汽车总利润池的物理上限）+ 三情景。
- 零部件：EV/EBITDA + 订单簿覆盖；电动化敞口（ICE 依赖收入占比）作为终值折价因子。
- 残值/召回/价格战风险用情景概率，不平均化。

## 4. 数据源与财报模式

1. 各国上险量/注册数据（中国交强险、欧洲 ACEA、美国注册）优于公司自报交付。
2. 财报重点：单车毛利环比（剔积分）、库存天数、价格行动、产能爬坡、FCF（成长期公司现金比利润重要一个量级）。
3. 月度销量高频跟踪；价格战监测（终端折扣数据）。

## 5. 红旗与输出要求

- 红旗：批零剪刀差扩大（压库）、应付账款周期异常拉长（拿供应商当融资）、监管积分依赖、研发资本化激进、创始人质押。
- 输出：单车经济表（ASP/成本/毛利逐季）+ 盈亏平衡销量测算必备；EV 标的必须给现金 runway（当前烧钱率下够烧几个季度）。
