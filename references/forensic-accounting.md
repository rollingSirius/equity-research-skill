# 财报质量核查（Forensic Accounting）

目的：在估值之前先回答一个更基本的问题——**这份利润是不是真的**。本模块产出一个「财报可信度等级（A/B/C/D）」，直接接入 `valuation-methods.md` 第 8 节的否决项：等级 C 动作最高只到"观望"，等级 D 一律"规避"并在第一章显式警示。

执行纪律：所有可计算指标由 `scripts/check_research_output.py --financials` 自动复核（列名要求见第 7 节）；无法取数的项目写"未获取到"，不许凭印象打分。

## 1. 应计质量（Accruals，Sloan 异象）

盈利 = 现金 + 应计。应计占比越高，盈利质量越低、未来回报越差（Sloan 1996）。

- **总应计比率 = (净利润 − 经营现金流) ÷ 平均总资产**。经验参考：< 0（现金利润超过账面利润）为优；0–5% 正常；**> 10% 红旗**。
- **现金转化率 = CFO ÷ 净利润**，看 3–5 年趋势。长期 < 80% 且持续下滑必须解释；利润增长而现金转化恶化是核心风险，不是附注。
- 应计的分解：应收、存货、应付、递延的变化各贡献多少；一次性大额应计（如收购）单列，不污染趋势。

## 2. Beneish M-Score（盈余操纵概率）

八变量模型，比较本年（t）与上年（t-1）：

```text
M = −4.84 + 0.92·DSRI + 0.528·GMI + 0.404·AQI + 0.892·SGI
    + 0.115·DEPI − 0.172·SGAI + 4.679·TATA − 0.327·LVGI
```

| 变量 | 含义 | 计算 | 危险方向 |
|---|---|---|---|
| DSRI | 应收/收入指数 | (应收t/收入t)÷(应收t-1/收入t-1) | ↑ 提前确认收入 |
| GMI | 毛利率指数 | 毛利率t-1 ÷ 毛利率t | ↑ 毛利率恶化诱发操纵 |
| AQI | 资产质量指数 | 非流动非PP&E资产占比之比 | ↑ 费用资本化 |
| SGI | 收入增长指数 | 收入t ÷ 收入t-1 | ↑ 高增长公司动机更强 |
| DEPI | 折旧率指数 | 折旧率t-1 ÷ 折旧率t | ↑ 放慢折旧 |
| SGAI | 费用率指数 | SGA率t ÷ SGA率t-1 | ↑ 效率恶化 |
| TATA | 总应计/总资产 | (净利润−CFO)÷总资产 | ↑ 权重最大项 |
| LVGI | 杠杆指数 | 杠杆t ÷ 杠杆t-1 | ↑ 违约压力 |

判读：**M > −1.78 → 落入"可能操纵"区**（原模型阈值）。这不是定罪，是把该公司移入"逐项手工核查"名单。金融股不适用（资产负债结构不同），改用第 6 节行业替代项。

## 3. 收入确认红旗

- **DSO 背离**：应收增速持续 > 收入增速（DSO 连续 2 年以上抬升）→ 渠道塞货 / 放宽信用 / 提前确认。
- **递延收入 / 合同负债背离**（订阅类致命项）：收入加速而递延收入减速甚至下降 → 增长在透支未来。
- 长期合同（POC 法）、bill-and-hold、经销商压货、与关联方的循环交易、季末突击发货（Q4 占比异常）。
- 重述历史、更换收入确认政策、分部重划导致不可比 → 一律建桥，桥不平就是红旗。

## 4. 费用资本化与利润平滑

- 资本化开发支出 / 软件 / 客户获取成本占比及其趋势；**capex 长期 > 2× 折旧摊销**而资产周转率没有相应提升 → 可能把费用藏进资产负债表。
- 存货细分：原材料 vs 产成品增速背离（产成品堆积 = 需求恶化的领先指标）。
- 减值的节奏：牛市不减、换 CEO 集中"洗大澡"（big bath）；准备金（坏账/保修/退货）计提率是否逆周期波动（利润平滑工具）。
- "非经常性损益"连续 3 年以上出现 → 重分类为经常性，直接调整常态化盈利。

## 5. 结构与治理信号（非量化，但权重高）

- 审计师：非四大/频繁更换/审计费异常低、内控重大缺陷（SOX 302/404）、非标意见。
- 人事：CFO/审计委员会主席突然离职；财报发布推迟。
- 监管：SEC comment letter / 交易所问询函的主题与频率；做空报告的核心指控与公司回应质量。
- 关联交易：定价公允性、体外循环（供应商/客户与实控人关联）；A股另查股权质押比例与资金占用（见 `markets-cn-hk.md`）。
- 激励结构：以调整后指标（尤其剔除 SBC 的 Non-GAAP EPS）为考核基准 → 操纵动机内建。

## 6. 行业替代项

金融/保险不用应计与 M-Score，改查：贷款分类迁徙与重组贷款、拨备覆盖率逆周期变化、金融资产三级（Level 3）占比、准备金释放对利润的贡献（保险）、再保险安排。REIT 查 AFFO 调整项与资本化利息。能源查储量重述与 DD&A 口径。

## 7. 检查器接入（列名约定）

`check_research_output.py --financials` 在 CSV 含以下列时自动计算并输出 P1/P2 问题：

- 必需（应计比率）：`net_income`、`cfo`、`total_assets`
- M-Score 另需：`revenue`、`receivables`、`gross_profit`、`ppe`、`current_assets`、`depreciation`、`sga`、`total_liabilities`
- DSO/递延背离另需：`receivables`、`deferred_revenue`（可选）

## 8. 财报可信度评级（预注册）

| 等级 | 判定标准 | 对结论的强制作用 |
|---|---|---|
| A | 无红旗；现金转化 ≥ 90%；应计 < 5%；治理干净 | 无限制 |
| B | 1–2 项轻度红旗（如单年应计偏高）且有合理解释 | 无限制，但列入监控指标 |
| C | M-Score 越限、DSO/递延持续背离、经常性"非经常"、治理红旗 1 项严重 | **动作最高"观望"**，报告第一章须警示 |
| D | 多项严重红旗叠加 / 涉嫌造假证据 / 审计非标 | **规避**，估值章节改为"估值在可信度恢复前无意义" |

评级和逐项证据写入报告第五章"财报质量"小节；等级 C/D 时禁止用"估值便宜"对冲——便宜是造假最常见的伪装。
