# 报告模板：九章结构（v2）

严格按以下章节顺序输出。**撰写前先读 `references/output-format.md`**（结论框/Tearsheet/本章要点/数字规范/football field 画法均在其中）。没有的数据写"未获取到"，不要留空表头。

格式约定：标题 `# 公司名（代码）个股投资研究报告`，副标题标"撰写日期 / 数据截止日 / 报告币种"。A/H 标的第六章末附两地对比、第一/九章分市场结论。

---

## 一、一页速览（Executive Summary）

依次包含（顺序固定）：

1. **结论框**（引用块）：标签 + 动作 + 财报可信度等级 + 置信度 + 一句话论点 + 综合区间与隐含空间 + 最大风险。
2. **Tearsheet 快照表**：现价/市值/52周/关键倍数/护城河/可信度/催化剂 Top1/证伪信号 Top1（每行来源+时间戳）。
3. **预期差 Gap 表**（本模板的脊柱，骨架见 `expectations-investing.md`）：市场隐含 vs 我的预期 vs base rate 分位，净预期差方向。
4. 核心多空逻辑各 3 条（精炼、可证伪）。

## 二、业务详情

本章要点先行。商业模式拆解、收入结构（业务线/地区/客户，占比+增速表）、集中度、产业链位置。

| 业务线/分部 | 最近期收入 | 占比 | 同比 | 备注 |
|---|---|---|---|---|

## 三、业务与竞争分析

行业空间与周期位置；竞争格局与份额趋势；**护城河评分表**（无形资产/转换成本/网络效应/成本优势/规模，各给 有无+强度+依据，综合判 无/窄/宽）——须与第六章 EPV/净资产、EVA 的财务验证交叉印证，不一致处显式解释；增长引擎与证伪点。

| 股权稀释 | SBC/收入、净稀释率 | | |

## 五、财务分析与财报质量

**5.1 财务趋势**：近 5 年营收/毛利率/经营利润率/FCF/ROIC/ROE 五年表 + 资产负债表健康度 + 同业对比表（沿用原骨架）。

**5.2 财报质量（forensic，必做）**：按 `forensic-accounting.md` 执行——应计比率与现金转化趋势、M-Score（可算则算）、DSO/递延背离、资本化政策、治理信号；输出**财报可信度等级 A/B/C/D** 及逐项证据表：

## 六、估值（多方法交叉验证）

按 `valuation-methods.md` 执行，**至少三种**，顺序建议：

1. **反向 DCF + PVGO**（开篇：现价隐含什么，对照 base rates）。
2. 三情景概率加权 DCF（+ 可选蒙特卡洛分布）。
3. 三要素/EPV（含护城河财务验证）。
4. EVA/剩余收益（含 g = RR×ROIIC 自洽检验）。
5. 相对估值（应得倍数纪律）/ SOTP / 行业特定法。

输出：各方法假设表（附 base rate 分位）→ **估值汇总表 + 文本版 football field 图** → WACC×g 敏感性（高于现价的格加粗）→ 综合区间。所有数字出自 `scripts/dcf.py`，假设 JSON 留档。

## 七、分析师评价汇总

评级分布/目标价区间与中位数/近期调整；**共识 vs 我的分歧点**（与第一章 Gap 表呼应——这是报告最有价值的部分）。

## 八、最新新闻与催化剂

近 1–3 个月要事；未来 6–12 个月催化剂时间表（区分已确认日程/管理层目标/市场预期），标多空方向与影响逻辑。

| 时间 | 事件 | 多/空 | 影响逻辑 | 确认状态 |
|---|---|---|---|---|

## 九、投资结论、红队与仓位

**9.1 结论**：标签+动作（与第一章一致，由标定规则映射）；回答"如果今天这是一笔现金，我会买入它吗？为什么？"

**9.2 红队/Pre-mortem（必做）**：换到反方立场，写出"一年后这笔投资失败了，最可能的 3 个原因"；对每条给出当前证据强度与监控信号。红队不是免责套话——至少一条必须是对本报告核心论点的直接攻击。

| 失败情景 | 触发机制 | 当前证据强度 | 监控信号 | 证伪时点 |
|---|---|---|---|---|

**9.3 仓位思维**：期望收益 EV、上行/下行不对称比、P(loss)、Kelly-lite 量级（小/中/标准仓位；非配置建议）。

**9.4 监控清单**：3–5 个持续跟踪指标（KPI/估值锚/证伪信号），各给关注阈值。

**9.5 置信度自评表**（骨架见 `output-format.md` 第 7 节）。
