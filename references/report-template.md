# 报告模板：九章结构

严格按以下章节顺序输出。结论先行、逻辑递进；关键数据用表格。下面每节给出"应包含的内容"与"表格骨架"，按标的实际情况填充；没有的数据写"未获取到"，不要留空表头。

格式约定：报告标题用 `# 公司名（代码）个股投资研究报告`，副标题标注"撰写日期 / 数据截止日"。

---

## 一、一页速览（Executive Summary）

- 一句话生意本质（这家公司靠什么赚钱）。
- 当前价格 / 市值 / 关键估值倍数（P/E、EV/EBITDA、P/S 等，标时间戳）。
- 核心多空逻辑各 3 条（精炼、可证伪）。
- **明确结论**：低估 / 合理 / 高估 + 建议动作（买入 / 观望 / 规避 / 减持）。

## 二、业务详情

- 商业模式拆解：做什么生意、靠什么赚钱、收入确认方式（一次性/订阅/授权等）。
- 收入结构：按业务线 / 产品 / 终端市场 / 地区 / 客户拆分，附占比与增速。
- 主要客户与上下游、收入集中度（前几大客户占比）。
- 所处产业链位置与价值量分布。

## 三、业务与竞争分析

- 行业空间（TAM / SAM）、增长驱动与周期位置。
- 竞争格局与公司市场地位、份额变化趋势。
- **护城河分析**：成本优势 / 网络效应 / 转换成本 / 无形资产（专利、技术、品牌）/ 规模——逐项判断是否存在，并给出**强度判断（无/窄/宽）**与依据。
- 增长引擎：未来 3–5 年核心增量来自哪里。
- 主要风险与潜在证伪点：技术、竞争、监管、客户集中、宏观/周期。

## 四、管理层与公司治理

- 核心高管背景、任职年限、过往业绩记录。
- 内部持股比例、激励机制是否与股东利益一致。
- 资本配置历史：回购 / 分红 / 并购 / 再投资及其效果评价（是否创造价值）。
- 治理红旗（若有）：关联交易、股权激励稀释、信披问题、双层股权等。

## 五、财务分析

- 近 5 年关键指标趋势：营收、毛利率、经营利润率、自由现金流、ROIC / ROE。
- 资产负债表健康度：现金、有息负债、净负债/EBITDA、利息覆盖倍数。
- 盈利质量：经营现金流与净利润匹配度、应收/存货是否异常、股权激励占比。
- 与同业横向对比。

## 六、估值（多方法论交叉验证）

按 `valuation-methods.md` 操作，**至少三种方法**，每种给区间。

- 相对估值：P/E、EV/EBITDA、P/S、PEG，与自身历史区间及可比公司对比。
- 绝对估值：DCF，列出关键假设（增速、利润率、WACC、永续增长）并做敏感性分析。
- 其他适用方法：分部估值（SOTP）、Morningstar 公允价值、或行业特定指标（如 ARR 倍数、单位经济）。
