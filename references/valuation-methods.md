# 估值方法

目标：用**至少三种**互相独立的方法给出合理价值区间，最后汇总成一张表，并算出相对当前股价的隐含上行/下行空间。多方法的意义在于交叉验证——若三种方法指向相近区间，结论更可信；若分歧很大，要解释为什么（通常是增长/盈利假设差异）。

原则：
- 每种方法都**显式列出关键假设**，假设比小数点更重要。
- 区分"当前倍数"与"合理倍数"，说明合理倍数的依据（历史中枢、可比公司、增长质量）。
- 估值是区间不是点，给保守/中性/乐观三档更好。

---

## 1. 相对估值（可比公司 + 历史区间）

常用倍数，按公司类型选择：
- **P/E（TTM 与 Forward）**：盈利稳定的公司。
- **EV/EBITDA**：资本结构差异大、折旧重的公司，跨公司更可比。
- **P/S**：高增长但未稳定盈利的公司。
- **PEG = P/E ÷ 盈利增速**：成长股，>1 偏贵、<1 偏便宜（粗略）。
- 高增长软件/订阅：**EV/Sales、EV/ARR**；亦看 Rule of 40。

做法：
1. 取公司自身**历史倍数区间**（如近 5 年 P/E 的 25–75 分位与中位），看当前处于什么位置。
2. 选 3–6 家**可比公司**，对比同一倍数。
3. 给出"合理倍数 × 对应财务指标"= 每股合理价值区间。

相对估值表：

| 倍数 | 当前 | 自身历史中枢 | 可比公司中位 | 选用合理倍数 | 隐含每股价值 |
|---|---|---|---|---|---|

## 2. 绝对估值：DCF

两段式自由现金流折现（显性期 5–10 年 + 永续值）。

关键假设（**必须列出**）：
- 收入增速路径（逐年，反映周期/渗透/见顶）。
- 经营利润率/FCF 利润率路径（向稳态收敛）。
- **WACC**：用 CAPM 估权益成本（无风险利率 + β×风险溢价），按资本结构加权；写明取值。
- **永续增长率 g**：一般 2–3%（不超过长期 GDP/通胀），高于此需强论证。
- 税率、再投资/资本开支假设。

输出：企业价值 → 减净债 → 股权价值 → ÷ 股本 = 每股内在价值。

**敏感性分析（必做）**：至少做 WACC × 永续增长 的二维表；成长股建议再做一张 收入增速 × 稳态利润率 表，因为它们对估值最敏感。

| WACC \ g | g-1% | g | g+1% |
|---|---|---|---|

| WACC | | | |
| WACC+1% | | | |

## 3. 其他适用方法（按行业选其一以上）

- **SOTP 分部估值**：多业务公司，对每个分部用各自合适的倍数/DCF 估值后加总，减净债与总部成本。适合分部盈利能力/增速差异大的公司。
- **Morningstar 公允价值**：作为独立第三方锚点直接引用（标日期与星级、不确定性评级）。
- **行业特定指标**：
  - 半导体/周期：归一化（mid-cycle）盈利 × 周期中枢倍数，避免在周期顶/底用当期 EPS。
  - SaaS/订阅：EV/ARR、LTV/CAC、净收入留存（NRR）。
  - 银行/保险：P/B 与 ROE 联动、P/Embedded Value。
  - 资源/地产：NAV/重置价值。

收尾：明确给出 低估 / 合理 / 高估 的判断。注意估值只是决策的一部分——要与护城河、增长确定性、管理层和风险一起看。当输入假设不足以支撑可信区间时，诚实写"无法给出可靠估值区间"，并说明缺什么数据。
