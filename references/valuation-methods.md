# 估值方法

目标：用**至少三种**互相独立的方法给出合理价值区间，汇总成表并算出相对现价的隐含空间。多方法的意义在于交叉验证——区间相近则结论更可信；分歧大要解释为什么（通常是增长/盈利假设或折现率差异），并说明采信哪个。

原则：
- 每种方法**显式列出关键假设**，假设比小数点重要；关键假设旁标注 base rate 分位（见 `base-rates.md`）。
- **所有 DCF/EPV/EVA/蒙特卡洛计算一律用 `scripts/dcf.py` 执行，禁止心算**。假设以 JSON 落盘，本身就是留档。
- 折现率全报告同源同值，构建方法见 `cost-of-capital.md`；风险只惩罚一次（折现率或情景概率，二选一）。
- 估值是区间不是点；结论标签由第 9 节的预注册规则映射产出，不允许临场发挥。
- **估值前置条件**：财报可信度等级（`forensic-accounting.md`）为 C/D 时，估值让位于可信度问题。

## 1. 相对估值（可比公司 + 历史区间 + 合理倍数）

常用倍数按公司类型选择：P/E（盈利稳定）、EV/EBITDA（重资产/资本结构差异大）、P/S（未盈利成长）、PEG（粗略）、EV/ARR（订阅）、P/B–ROE（金融）。

做法：
1. 公司自身**历史倍数区间**（近 5–10 年 25/50/75 分位），看当前位置。
2. 3–6 家**可比公司**同一倍数对比（A股/港股对照系见 `markets-cn-hk.md` 第 6 节）。
3. **合理倍数（warranted multiple）纪律**：不许直接抄同业中位数。倍数是增长 g、回报 ROIC、风险 r 的函数——稳态下合理 P/E ≈ (1 − g/ROIIC)/(r − g)。先算公司基本面对应的合理倍数，再与市场给的倍数比较；差额要么是市场错（机会），要么是你没看到的风险（警报），显式归因。
4. "合理倍数 × 对应财务指标" = 每股合理价值区间。

| 倍数 | 当前 | 自身历史中枢 | 可比中位 | 合理倍数(算式) | 选用 | 隐含每股价值 |
|---|---|---|---|---|---|---|

## 2. 绝对估值：DCF（正向）+ 终值纪律

三阶段：显性期 5–10 年 + 增速衰减期 3–6 年 + 永续。衰减期让末年增速线性降至永续 g，避免断崖。

关键假设（必须列出，附 base rate 分位）：收入路径、FCF 利润率路径、WACC（见 `cost-of-capital.md`）、永续 g（≤长期名义 GDP）、股本稀释率。

**终值纪律（终值合理性三查）**：
1. 隐含退出 P/FCF（脚本自动输出）≤ 成熟同业当前倍数，超出须论证。
2. 终值占总价值 > 80% → 说明显式期太短或市场在买遥远故事，明示。
3. **默认让增量回报向资本成本收敛**：永续期新增投资 ROIIC → WACC（增长不再增值，即永续期价值≈EPV 式零增长价值）。要在终值里保留超额回报，必须以护城河持续期（CAP）显式论证——历史上超额回报期 >15–20 年的公司极少（`base-rates.md` 第 4 节）。

执行：假设 JSON → `scripts/dcf.py --config`。敏感性 WACC×g 3×3 必做；成长股加做 收入增速×稳态利润率。

## 3. 三要素估值法：资产重置 · EPV · franchise 成长（Graham–Greenwald）

（方法学不变，与旧版一致；与本手册其他方法冲突时以 Greenwald 原著为准。）

**第一层 · 资产重置价值（底价）**：资产负债表逐项挤水分减总负债。折扣参考：现金 100%、证券 95–100%、应收 80–90%、存货（原料/成品）80–90%/50–70%、设备 20–30%、厂房 40–60%、地产按市价、商誉多按 0。

**第二层 · EPV（核心）**：`EPV = 常态化盈利 ÷ 资本成本`（零增长永续）。常态化盈利剔非经常、加回超维持性 capex 的 growth capex；默认 NOPAT 口径（÷WACC 得 EV，减净债加过剩现金得权益）；数据不足退净利润口径（coc 视作权益成本）。资本成本与 DCF 的 WACC 同源。**护城河财务验证**：EPV/调整后净资产的多年趋势——≈1 无护城河、<1 毁灭价值、≫1 且长期稳定 = 护城河的财务度量；与第三章定性评分交叉印证。

**第三层 · 成长价值（仅护城河内计）**：`价值 = E×(1 − g/ROIIC)/(coc − g)`，要求 ROIIC>coc 且 g<coc；否则退简化式 EPV×(1+保守 g)。保守 g = 历史 5 年利润 CAGR 与宏观锚取低。

执行：JSON `epv` 块 → 脚本输出买点阶梯（底价 < EPV < 成长调整）与现价所处区间。

## 4. 反向 DCF + PVGO 分解——预期差分析主线（第一章开篇框架）

反向 DCF 问"**现价假设了什么**"，是整份报告的分析主线（完整方法论见 `expectations-investing.md`）。

- `reverse` 块：由现价解出稳态所需 FCF → 按稳态利润率档换算所需营收与隐含 CAGR。
- `pvgo` 块：`现价 = E/r（零增长价值） + PVGO`，输出**现价中为未来增长付费的百分比**。PVGO 占比 >50% 的标的，报告必须回答"这些增长从哪来、谁买单"。
- 呈现：把隐含要求翻译成参照系（对照 base rates："隐含连续 N 年 Y% 增速，历史上做到的公司占 Z%"）。结论句式："市场不是在给 × 定价，而是在给 × 定价"。
- 产出**预期差 Gap 表**（骨架见 `expectations-investing.md` 第 2 节），置于报告第一章。

## 5. 情景加权公允价值 + 蒙特卡洛

**三情景（必做）**：熊/基准/牛各为一套内部自洽假设（不是基准 ±20%）；熊市要问"理性管理层届时会怎么做"。赋概率（和为 1）写依据；脚本一次完成加权。**稳健性检验**：把概率推到对结论最不利的组合，看标签是否翻转；翻转则如实写"结论对概率敏感"。

**蒙特卡洛（可选加强，结果分布很宽或路径依赖强时）**：`montecarlo` 块给关键驱动设分布（增速：正态截断；利润率：三角；WACC：均匀窄带），N≥2000 次模拟，输出公允价值 P10/P25/P50/P75/P90、**P(内在价值 < 现价)（即"现价买入是错误"的概率）**与期望值。蒙特卡洛不替代三情景（相关性设定不透明时反而伪精确），作为分布形状的补充证据。

## 6. EVA / 剩余收益（经济利润）——护城河的另一把尺

`EVA_t = NOPAT_t − WACC × 投入资本_{t-1}`；`价值 = 当前投入资本 + Σ 未来 EVA 折现`。

为什么值得单独做：把价值拆成"存量资本 + 超额回报"，直接暴露两个 DCF 里看不见的问题——
1. **公司现在到底赚不赚超过资本成本的钱**（EVA 连年为负的"增长股"是价值粉碎机）；
2. **增长的价值来源**：`内生增长 g = 再投资率 RR × ROIIC`——想要 10% 增长、ROIIC 只有 12%，意味着要再投 83% 的利润，FCF 所剩无几。DCF 里增速和利润率是两个自由参数，EVA 框架强迫它们自洽。

执行：JSON `eva` 块（invested_capital、nopat、wacc、fade_years：超额回报 Δ=ROIC−WACC 线性衰减至 0）→ 脚本输出 EVA 序列、剩余收益价值与每股。金融股用剩余收益的权益版（净利润 − CoE×净资产），是银行/保险的主力绝对估值法。

## 7. 其他适用方法

- **SOTP**：多业务公司分部加总，减净债与总部成本；与 DCF 的分歧=市场情绪的定量化。平台/集团必做。
- **Morningstar 公允价值**：第三方锚，标日期/星级/不确定性；区分分析师 FV 与量化评级。
- **行业特定**：NAV（矿业/航运/地产）、rNPV（医药管线）、P/EV（保险）、P/AFFO（REIT）、中周期盈利法（周期股，通用规则：中周期 EPS = 10 年利润率带宽中点 × 当前收入；禁止峰值×峰值）。

## 9. 结论标定与仓位思维（预注册，不得临场发挥）

**估值标签**（综合区间 [L, H]，现价 P，±15% 缓冲带）：

| 条件 | 标签 |
|---|---|
| P < 0.50L | 显著低估 |
| 0.50L ≤ P < 0.85L | 低估 |
| 0.85L ≤ P ≤ 1.15H | 合理 |
| 1.15H < P ≤ 1.50H | 高估 |
| P > 1.50H | 显著高估 |

**动作矩阵**（标签 × 不确定性；不确定性优先 Morningstar 评级，缺失自评并写依据）：

| 标签 \ 不确定性 | 低 / 中 | 高 / 极高 |
|---|---|---|
| 显著低估 | 买入 | 分批买入（P < 0.70L 方可首建） |
| 低估 | 买入/分批买入 | 小仓位试探或等待 |
| 合理 | 观望（持有者持有） | 观望 |
| 高估 | 减持/规避 | 规避 |
| 显著高估 | 规避，持有者减持 | 同左；默认不建议做空 |

**否决项（一票降级）**：治理红旗 ≥2 项严重级或**财报可信度 C** → 动作最高"观望"；**可信度 D** → 规避；护城河"无"且标签高估 → 直接规避。双重上市按 `markets-cn-hk.md` 分市场标定。

**仓位思维（`position` 块，给动作配量级）**：
- 期望收益 `EV = Σ p_i × (V_i/P − 1)`（用三情景的每股价值与概率）。
- **不对称比 = 上行空间 ÷ 下行空间**（牛情景涨幅 ÷ 熊情景跌幅）；<1.5 的"低估"多半不值得建仓。
- Kelly-lite 参考仓位 = ¼ × Kelly 分数（脚本按两点近似计算），仅作量级参考、上限 15%，**不构成配置建议**——报告只说"该赔率结构支持 小/中/标准 仓位"。
- 亏损概率 P(loss)（蒙特卡洛可得）与最大可容忍下行一并呈现。

**一致性要求**：第一章与第九章的标签、动作、预期差方向必须与本规则可复算地一致；`dcf.py` 在给定 range_low/range_high 时自动输出标签建议，`check_research_output.py` 复核报告文本一致性。
